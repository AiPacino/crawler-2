平安银行如何登录网上银行？

一、网银导入只支持网上银行的账号登录

    本平台的网银导入服务只支持平安银行个人网上银行登录，即您注册该网银的身份证号码、一账通用户名、手机号，不支持手机银行登录和企业网银登录。

    平安银行个人网上银行官网链接：https://b.pingan.com.cn/


二、如何确认自己登录账号、登录密码的有效性

    1、请访问平安银行官网（https://b.pingan.com.cn/ ），点击【账户余额】，如下图所示。


    2、在打开的页面中进行登录验证。如果能登录成功，请将相应登录账号和登录密码在本平台进行登录即可完成网银导入。


三、如何注册平安银行网上银行

    1、如果您持有平安银行的储蓄卡，请点击【快速注册】直达链接：https://bank-static.pingan.com.cn/ibank/member/mobile/register/pages/register.html?__paramed__。（如下图）


    2、在打开的页面填写相关信息即可完成注册。

    3、注册完毕后，请访问平安银行官网（https://b.pingan.com.cn/ ），点击【账户余额】，使用刚刚注册的手机号和查询密码进行登录，再点击【快速添加银行卡】，填写相关信息完成绑卡操作（只支持绑定储蓄卡，如下图），最后再使用本平台进行网银导入。


四、我使用U盾、数字证书登录网银怎么办

    目前本平台暂不支持这类网银导入，建议您按照上面的开通网银教程，自助开通平安银行个人网上银行，如有疑惑，可致电平安银行客服：95511 咨询。


五、忘记网银登录密码怎么办

    请点击【忘记密码】直达链接：https://bank-static.pingan.com.cn/ibank/member/mobile/forget-password/pages/find-password.html?__paramed__，进行重置密码操作。


六、常见问题

    Q：提示【用户名或密码错误】，怎么办？

      1、请确认您的网上银行（与手机银行不同）已经开通并激活，您可以联系客服热线（95511）确认是否已经成功注册网上银行，如若未注册网银，请参照上面的教程进行注册👆

      2、如果已在柜台开通网银，请确认您输入的账号有效，建议您可以在官网（https://b.pingan.com.cn/），点击【账户余额】，进行登录验证，如若忘记密码，请直接访问平安银行网上银行，点击【忘记密码】进行重置密码操作。

      3、如果能在官网正常登录，仍然无法正常导入，可能是服务器繁忙，您可以稍后再试。

    Q：提示【密码试错超限】，怎么办？

      平安银行网上银行一天内输错密码的次数是有限的，借记卡累计输错5次／信用卡累计输错5次，网银将会被锁住24小时，建议您联系客服热线95511，或确认密码后（24小时后）再进行尝试登录。

    Q：未收到【短信验证码】，怎么办？

      1、请确认当前手机号是开通网上银行时填写的手机号，如若不是，请携带相关证件、银行卡等物品到邻近网点办理更换手机号业务。

      2、短信发送时可能存在延迟现象，请您耐心等待。

      3、如果确认当前手机号是开通网上银行时填写的手机号，仍无发正常发送短信验证码，可能是发送短信过于频繁，您可以稍后再试。
