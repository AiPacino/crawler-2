北京银行如何登录网上银行？

一、网银导入只支持网上银行的账号登录

    本平台的网银导入服务只支持北京银行网上银行登录，即您注册该网银的信用卡卡号&证件号码，不支持手机银行登录和企业网银登录。

    北京银行网上银行登录入口：https://ebank.bankofbeijing.com.cn/bccbpb/accountLogon.jsp?language=zh_CN


二、如何确认自己登录账号、登录密码的有效性

    请访问北京银行个人网银官网（https://ebank.bankofbeijing.com.cn/bccbpb/accountLogon.jsp?language=zh_CN）进行登录验证。如果能登录成功，请将相应登录账号和登录密码在本平台进行登录即可完成网银导入。


三、忘记网银登录密码怎么办

    如果您遗忘网银登录密码，请携带本人有效身份证件和办理网银时使用京卡或存折，到网银开户网点办理密码重置手续。密码重置后恢复为6个零的初始密码，您登录网银后系统将强制您修改密码。


四、常见问题

    Q：提示【用户名／密码错误】，怎么办？

      1、请确认您的网上银行（与手机银行不同）已经开通并激活，您可以联系客服热线（95526）确认是否已经成功开通网上银行，如若未开通网银，请携带相关证件、银行卡等物品到邻近网点开通网上银行服务。

      2、如果已在柜台开通网银，请确认您输入的账号有效，建议您可以在官网（https://ebank.bankofbeijing.com.cn/bccbpb/accountLogon.jsp?language=zh_CN）进行登录验证，如若忘记密码，请携带本人有效身份证件和办理网银时使用京卡或存折，到网银开户网点办理密码重置手续。密码重置后恢复为6个零的初始密码，您登录网银后系统将强制您修改密码。

      3、如果能在官网正常登录，仍然无法正常导入，可能是服务器繁忙，您可以稍后再试。

    Q：提示【密码试错超限】，怎么办？

      北京银行借记卡一天内输错密码的次数是有限的，累计输错6次，网银将会被锁住，建议您联系农行客服热线95526，或第二天确认密码后再进行尝试登录。
