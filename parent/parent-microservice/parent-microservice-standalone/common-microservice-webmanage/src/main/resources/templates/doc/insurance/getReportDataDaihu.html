获取社保报告-贷乎

说明

调用该接口可以根据爬虫任务ID获取获取用户的社保报告。

URL

http://<project>/h5/insurance/getReportDaihu

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
shebaoTaskId false String 社保唯一标识

注意事项

例子

说明：调用本接口，获取用户的社保报告。

请求：https://uatapitxboss.txtechnologies.com:9018/h5/insurance/getReportDaihu

返回成功：JSON
