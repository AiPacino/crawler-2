淘宝账号登录接口

说明

该接口通过输入的会员名、邮箱、手机号登录淘宝官网，返回是否成功信息，数据格式为JSON

URL

http://<project>/e_commerce/taobao/alipay/loginAgent

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
idnum true String 用户身份证号
username true String 用户名
passwd true String 登录密码
logintype true String 登录类型
verfiySMS false Integer 手机短信验证码
taskid true Integer 任务id

注意事项

    1.手机号必须为11位手机号。
    2.登录类型只有"sn_num"（账号登录）和"sn_phone"（短信登录）两种。

例子

说明：账号登录苏宁官网

请求：http://10.167.202.217:10008/ds/suning/login

返回：JSON
