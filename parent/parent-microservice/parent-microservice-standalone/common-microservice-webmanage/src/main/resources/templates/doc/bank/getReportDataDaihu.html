获取网银报告-贷乎

说明

调用该接口可以根据爬虫任务ID获取获取用户的网银报告。

URL

http://<project>/h5/bank/getReportDaihu

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
wangyinTaskId true String 网银唯一标识

注意事项

例子

说明：调用本接口，获取用户的网银报告。

请求：https://uatapitxboss.txtechnologies.com:9018/h5/bank/getReportDaihu

返回：JSON

JSON示例

 
{
    
    "onlineBankDetail": {
					    "tradeTime":"2018-01-01 23:22:00",
					    "payment":"100",

					    "abstracts":"自动还款",

					    "tradePlace":"网上银行",
					    "otherAccount":"***********",
					    "otherAccountName":"张三",
					    "otherAccountBank":"工商银行",
					    "bankCard":"************1564",

}

返回字段

对象名 字段名 类型 注释
onlineBankDetail（网银数据信息） tradeTime String 交易时间
onlineBankDetail（网银数据信息） payment String 支出
onlineBankDetail（网银数据信息） deposit String 收入
onlineBankDetail（网银数据信息） abstracts String 摘要
onlineBankDetail（网银数据信息） balance String 余额
onlineBankDetail（网银数据信息） tradeType String 交易类型
onlineBankDetail（网银数据信息） tradePlace String 交易地点
onlineBankDetail（网银数据信息） otherAccount String 对方账号
onlineBankDetail（网银数据信息） otherAccountName String 对方账号名称
onlineBankDetail（网银数据信息） otherAccountBank String 对方开户行
onlineBankDetail（网银数据信息） bankCard String 当前卡所属人银行卡号
onlineBankDetail（网银数据信息） orderIndex String 排序
onlineBankParent（网银父级数据信息） bankName String 银行名称
onlineBankParent（网银父级数据信息） bankCard String 当前卡所属人银行卡号
onlineBankParent（网银父级数据信息） userName String 当前所属人名称
onlineBankParent（网银父级数据信息） reportResult String 报告结果
onlineBankParent（网银父级数据信息） reportUrl String 报告地址
