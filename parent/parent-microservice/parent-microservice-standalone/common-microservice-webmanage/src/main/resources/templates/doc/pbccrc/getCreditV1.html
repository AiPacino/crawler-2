获取征信报告

说明

调用该接口可以登录人行征信系统并获取用户的网页版征信报告。

URL

http://<project>/h5/pbccrc/v1/getCredit

格式

JSON

HTTP请求方式

POST

请求参数

# 必选 类型及范围 说明
username true String 用户名
password true String 登录密码
tradecode true String 手机验证码

注意事项

例子

说明：调用本接口，登录人行征信系统，并获取用户的网页版征信报告。
