苏宁统一数据接口

说明

该接口通过输入的登录名或taskid来获取统一处理过的苏宁爬取数据。

URL

http://10.167.202.203:8009/ecommerce/tasks/suning/detail

格式

JSON

HTTP请求方式

POST/GET

请求参数

# 必选 类型及范围 说明
loginName true String 登录名
taskid true String 任务号

注意事项

  1. 登录名或taskid必须有一个，如果taskid不为空，则默认用taskid查询，如果taskid为空，则用登录名查询，获取该登录名最新的数据

例子

说明：利用taskid和登录名来获取完成数据

请求：http://10.167.202.203:8009/ecommerce/tasks/suning/detail?taskid=9b560bcf-aa57-41c2-84c7-e1623fa790c4

返回：JSON

505 taskid not matches idnum! 任务号与身份证号不匹配
