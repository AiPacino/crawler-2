借么-异步状态

说明

该接口为推送爬虫的异步状态

格式

JSON

HTTP请求方式

POST

注意事项

例子

说明：调用借么的接口，推送异步状态

异步状态：JSON

JSON示例

{
	"key": "2005",
	"statusCode": "1",
	"message": "爬取成功"
}

异步状态码说明

异步状态码说明对照表：
码值 说明
-1 人行征信网站被屏蔽
-2 登录失败
-3 查询失败
0 登录成功
1 查询成功
2 征信报告未生成或授权码已过期
3 授权码错误
4 用户名或密码错误
5 系统繁忙，请稍后再试
6 网络超时，请重试
200 正在登录，请耐心等待...
300 正在查询，请耐心等待...
