借么-前置规则

说明

该接口为推送前置规则

格式

JSON

HTTP请求方式

POST

注意事项

例子

说明：调用借么的接口，推送前置规则，返回推送结果状态

前置规则：JSON

JSON示例

{
	"key": "123",
	"taskId": "f734cc88-5a5e-47da-b69a-4fb702814c46",
	"real_name": "徐玉君",
	"id_card": "**************0052",
	"reportTime": "2018.06.19 19:35:02",
	"type": "社保"
}

前置规则字段说明

前置规则字段说明对照表：
字段 字段类型 字段说明
key String 业务唯一标识
taskId String 爬虫任务ID
real_name String 真实姓名
id_card String 身份证号
reportTime String 社保的生成时间
type String 业务类型（征信，运营商、社保...）
