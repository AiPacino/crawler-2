获取爬取状态

说明

该接口通过传递的taskid查询是否完成爬取

URL

GET:http://10.167.202.187:51829/honesty/search/statue/taskid

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
taskid true String 业务id，区分每一次的业务

例子

说明：获取爬取状态

请求：http://10.167.202.187:51829/honesty/search/statue/taskid?taskid=0001

JSON示例

 
{
taskid: "0001",
totalnum: 1,
unfinishednum: 1,
finishednum: 0,
crawleringnum: 0,
errornum: 0,
errormessage: null
}
		
返回说明

				

                
                

                	字段名
                    解释说明
                

                
                
                

                    taskid
                    唯一标示
                

                

                    totalnum
                    爬取任务数量
                

                

                    unfinishednum
                    未完成数量
                

                

                    finishednum
                    完成数量
                

                

                    crawleringnum
                    正在爬取中的数量
                
  
                

                    errornum
                    出错的数量
                
 
                

                    errormessage
                    出错提示
