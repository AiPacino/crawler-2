借么-前置规则

说明

该接口为推送前置规则

格式

JSON

HTTP请求方式

POST

注意事项

例子

说明：调用借么的接口，推送前置规则，返回推送结果状态

前置规则：JSON

JSON示例

{
	"name_on_card": "张三",
	"idnum": "**************0338",
	"trans_date": "2018-1-1",
	"amount_money": "100",
	"category": "自动还款",
	"cardNo": "************5729",
	"cardId":"null"
}

前置规则字段说明

前置规则字段说明对照表：
字段 字段类型 字段说明
name_on_card String 持卡人姓名
idnum String 身份证号
trans_date String 交易时间
amount_money String 消费金额
category String 消费类型
cardNo String 卡号
cardId String 银行卡ID
