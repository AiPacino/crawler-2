添加关键词Post

说明

该接口通过传递的key，taskid，pagenum创建待爬取队列

URL

http://10.167.202.192:51317/sanwang/search/task

HTTP请求方式

POST

请求参数

# 必选 类型及范围 说明
keys true List<String> 传递的关键词
taskid true String 业务id，区分每一次的业务
pagenum true int 页码，爬取多少页

注意事项

  1. pagenum默认为1

例子

说明：创建爬取队列

请求：http:/10.167.202.192:51317/sanwang/search/task

JSON示例

 
{
taskid: "posttest23",
totalnum: 39,
unfinishednum: 0,
finishednum: 39,
crawleringnum: 0,
errornum: 1,
errormessage: null
}

返回说明

状态码 解释说明
taskid 唯一标示
totalnum 爬取任务数量
unfinishednum 未完成数量
finishednum 完成数量
crawleringnum 正在爬取中的数量
errornum 出错的数量
errormessage 出错提示
