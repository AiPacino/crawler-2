苏宁短信发送接口

说明

该接口通过输入的手机号，将短信登录验证码发送到手机号中，返回是否成功信息，数据格式为JSON

URL

http://<project>/suning/login

格式

JSON

HTTP请求方式

POST

请求参数

# 必选 类型及范围 说明
username true String 接收短信的号码
taskid true Integer 任务id

注意事项

    1.手机号必须为11位手机号。

例子

说明：发送登录短信验证码

请求：http://10.167.202.217:10008/ds/suning/sendSMS

返回：JSON

JSON示例

 
{
	"id": 855,
	"createtime": null,
	"taskid": "suning006",
	"phase": "WAIT_CODE",
	"phase_status": "SUCCESS",
	"description": "短信验证码已发送，请注意查收",
	"finished": false,
	"error_code": 0,
	"error_message": null,
}

返回字段

字段名 类型 注释
id String 唯一标识
createtime datetime 创建时间
taskid String 任务id
phase String 当前步骤
phase_status String 步骤状态
description String 描述
finished Boolean 爬虫任务是否全部完成
error_code String 错误代码
error_message String 错误信息

状态码映射

状态码 解释说明
0 成功
2 失败
