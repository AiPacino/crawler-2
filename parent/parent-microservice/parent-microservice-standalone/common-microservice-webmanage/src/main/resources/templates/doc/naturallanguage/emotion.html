情感倾向分析

说明

对包含主观观点信息的文本进行情感极性类别（积极、消极、中性）的判断，并给出相应的置信度。

URL

http://<10.167.202.217:>9357/nlp/emotion

格式

JSON

HTTP请求方式

POST

请求参数

# 必选 类型及范围 说明
text_emotion true String 文本内容

JSON示例:

 
   
       {
            "sentiment_emotion":2,    //表示情感极性分类结果
            "confidence_emotion":0.40, //表示分类的置信度
            "positive_prob_emotion":0.73, //表示属于积极类别的概率
            "negative_prob_emotion":0.27  //表示属于消极类别的概率
        }

返回字段

字段名 类型 注释
sentiment_emotion String 表示情感极性分类结果
confidence_emotion String 表示分类的置信度，取值范围[0,1]
positive_prob_emotion String 表示属于积极类别的概率 ，取值范围[0,1]
negative_prob_emotion String 表示属于消极类别的概率，取值范围[0,1]
