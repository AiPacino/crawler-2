获取登录页

说明

获取人行征信登陆页的cookie（注意不是登录成功后的cookie），以及进行登陆页的图片验证码识别。

URL

http://<project>/h5/v2/pbccrc/getLoginPage

格式

JSON

HTTP请求方式

GET

请求参数

无

注意事项

无

例子

说明：非调试模式下调用本接口，获取人行征信登陆页的cookie，并进行人行征信系统登录页图片验证码的识别

请求：https://uatapitxboss.txtechnologies.com:9018/h5/v2/pbccrc/getLoginPage

返回：JSON

返回字段

字段名 类型 注释
statusCode String 结果状态码
message String 结果状态说明
cookies Set<cookie> 登陆页cookie
codeImageUrl String 登陆页验证码图片URL
codeValue String 登陆页图片验证码识别结果

状态码映射

状态码 解释说明
0 成功
2 图片验证码错误
3 cookie错误
4 手机验证码错误
5 查询频率限制
6 缓存中数据
9 用户名或密码错误
-1 失败
