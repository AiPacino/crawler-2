获取社保报告-借么

说明

调用该接口可以根据爬虫任务ID获取获取用户的社保报告。

URL

http://<project>/h5/insurance/getReportJieMo

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
shebaoTaskId false String 社保唯一标识

注意事项

例子

说明：调用本接口，获取用户的社保报告。

请求：https://uatapitxboss.txtechnologies.com:9018/h5/insurance/getReportJieMo

返回成功：JSON

jinpoMessage->baseInfoPersonal String 信贷记录信息->民族
jinpoMessage->baseInfoUnitType String 信贷记录信息->单位类型
jinpoMessage->baseInfoSex String 信贷记录信息->性别
jinpoMessage->accAddr String 信贷记录信息->与身份信息姓名是否一致
jinpoMessage->idCardStatus String 社保数据信息->与身份信息身份证号码是否一致
jinpoMessage->accAddrStatus String 社保数据信息->户籍地址与身份信息地址是否一致
jinpoMessage->insureStatus String 社保数据信息->社保缴纳状态
jinpoMessage->liveAddr String 社保数据信息->现居住地址

jinpoMessage->cellphone String 社保数据信息->手机号码
jinpoMessage->phoneStatus String 社保数据信息->手机号码与个人信息手机号码是否一致
jinpoMessage->email String 社保数据信息->邮箱

jinpoMessage->payDateStauts String 社保数据信息->汇缴起始日期是否符合
jinpoMessage->baseInfoPersonnelStatus String 社保数据信息->人员状态(在职,离职，失业，复原)

insurePay->perRmb String 社保缴费记录表->个人缴费

insurePay->comRmb String 社保缴费记录表->单位缴费
insurePay->totalAmount String 社保缴费记录表->缴存总额
insurePay->isDelq String 社保缴费记录表->是否发生过逾期
socialSecurityCoverage->insuranceCode String 社保险种表->险种编号
socialSecurityCoverage->putsMonths String 社保险种表->缴存月数
socialSecurityCoverage->monthlyCorporationIncome String 社保险种表->公司缴存金额，单位为分
socialSecurityCoverage->ginsengTime String 社保险种表->本次参保时间
