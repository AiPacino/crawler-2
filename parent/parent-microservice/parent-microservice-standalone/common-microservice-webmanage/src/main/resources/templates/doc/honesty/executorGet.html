说明

该接口通过传递的keys，taskid创建待爬取队列

URL

http://10.167.202.187:51829/honesty//shixin/task

HTTP请求方式

POST

请求参数

实体名 字段名 必选 类型及范围 说明
HonestyJsonBean keys true list<Honestybean> 传递的关键词
HonestyJsonBean taskid true String 业务id，区分每一次的业务
HonestyJsonBean pagenum true Integer 查询页码
HonestyJsonBean pagesize true Integer 每页数量
Honestybean pName true String 传递的关键词
Honestybean pCardNum true String Honestybean
Honestybean pagenum true Integer 查询页码
Honestybean pagesize true Integer 每页数量

例子

说明：创建爬取队列

请求：http://10.167.202.187:51829/honesty/shixin/task

JSON示例

 
{
"taskid": null,

返回说明

状态码 解释说明
taskid 唯一标示
totalnum 爬取任务数量
unfinishednum 未完成数量
finishednum 完成数量
crawleringnum 正在爬取中的数量
errornum 出错的数量
errormessage 出错提示
