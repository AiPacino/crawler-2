公积金统一数据接口

说明

该接口通过输入的身份证号或taskid来获取统一处理过的公积金数据。

URL

http://10.167.202.203:8004/housing/detail

格式

JSON

HTTP请求方式

POST/GET

请求参数

# 必选 类型及范围 说明
idnum true String 身份证号
taskid true String 任务号

注意事项

  1. 身份证号或taskid必须有一个，如果taskid不为空，则默认用taskid查询，如果taskid为空，则用身份证号查询，获取该身份证号最新的数据

例子

说明：利用taskid和身份证号来获取完成公积金数据

请求：http://10.167.202.203:8004/housing/detail/?taskid=e5d82bae-582d-4405-b216-92b14e3f3562

返回：JSON
