第二次验证随机短信接口

说明

该接口通过taskId，发送第二次验证随机短信，数据格式为JSON

URL

http://<project>/sendVerifySMS

格式

JSON

HTTP请求方式

POST

请求参数

# 必选 类型及范围 说明
name true String 手机号
password true String 服务密码
sms_code true String 随机验证码
user_id true Integer 用户id
task_id true Integer 任务id

注意事项

  1. 手机号必须为11位手机号

例子

说明：发送第二次验证随机短信接口

请求：http://10.167.202.192:1002/sendVerifySMS

返回：JSON
