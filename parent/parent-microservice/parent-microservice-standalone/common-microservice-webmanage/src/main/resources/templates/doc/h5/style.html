设置风格

说明

该接口通过themeColor,设置H5页面的风格颜色

URL

http://<project>/h5/auth?themeColor=eda79a

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
themeColor false String 风格颜色

注意事项

  1. themeColor默认值为5bc0de
  2. themeColor的值不需要加 #

例子

说明：设置H5页面风格颜色

请求：https://uatapitxboss.txtechnologies.com:9018/h5/auth?themeColor=eda79a
