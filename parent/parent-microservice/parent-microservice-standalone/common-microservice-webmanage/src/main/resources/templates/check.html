身份验证接口

说明

该接口通过输入的手机号，返回手机号的运营商归属地，数据格式为JSON


URL

http://<project>/check


格式

JSON


HTTP请求方式

POST


请求参数

# 必选 类型及范围 说明
username true String 用户姓名
idnum true String 用户身份证号
mobileNum true String 11位手机号

注意事项

  1. 手机号必须为11位手机号，170等虚拟运营商可能无法获取手机号归宿地
  2. 移动部分省份（北京、湖北、内蒙古、四川、贵州、江西、山西、广东）手机号号码为特殊手机号,接口返回字段spec为true，电信、联通和其他地区移动运营商为false

例子

说明：获取手机号归宿地

请求：http://10.167.202.192:32796/check

返回：JSON

JSON示例

 
{
	"data": {
		"mobileNum": "18618135874",
		"mobileOperator": "UNICOM",
		"spec": false,
		"username": "梅荻",
		"idnum": "420106198410028419",
		"id": 2
	},
	"mesCode": "0000",
	"createTime": 1498720135365,
	"message": "查询完成"
}

返回字段说明

返回正确data结果的字段说明对照表：
返回值字段 字段类型 字段说明
mesCode String 结果状态码
createTime Integer 接口返回时间戳
message String 接口状态信息
data->mobileNum String 手机号
data->mobileOperator String 运营商
data->mobileNum String 手机号
data->spec Boolean 手机运营商归属地是否特殊
data->username String 用户姓名
data->idnum String 用户身份证号
data->id String 用户唯一ID
备注：
结果状态码（mesCode）说明：
码值说明
0000登录成功
1001随机短信发送失败
1002手机号认证失败(随机码不正确或已过期)
1003登录失败
