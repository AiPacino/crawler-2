获取网银报告-借么

说明

调用该接口可以根据爬虫任务ID获取获取用户的网银报告。

URL

http://<project>/h5/bank/getReportJieMo

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
wangyinTaskId true String 网银唯一标识

注意事项

例子

说明：调用本接口，获取用户的网银报告。

请求：https://uatapitxboss.txtechnologies.com:9018/h5/bank/getReportJieMo

返回：JSON
