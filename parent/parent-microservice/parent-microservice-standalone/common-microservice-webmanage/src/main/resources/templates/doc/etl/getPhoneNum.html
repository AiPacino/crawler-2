电话号码查询接口

说明

该接口通过输入身份证号码，来获取该身份证所属的所有电话号码。

URL

http://uat.admin.txboss.txtechnologies.com/data/carrier/tasks/getphonenum

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
idnum true String 身份证号

注意事项

  1. 身份证号必须合法，并不能为空

例子

说明：查询所属该身份证的所有电话号码

请求：http://uat.admin.txboss.txtechnologies.com/data/carrier/tasks/getphonenum?idnum=130984199210023312

返回：JSON

JSON示例

 
{
  "idnum": "130984199210023312",
  "phoneNumbers": [
    "18618135874",
    "18618135875"
  ],
  "message": "success",
  "createtime": 1503048363734,
  "errorCode": 800
}

返回字段

字段名 类型 注释
idnum String 身份证号
phoneNumbers String 电话号
message String 状态消息
errorCode String 状态码
createtime timestamp 当前时间戳

状态码映射

状态码 信息 解释说明
800 success 成功
807 ID is null,please check! 身份证号为空
808 there is no phonenumber according to this ID ! 该身份证号无对应电话号
809 this ID is not in database ! 该身份证在数据库中不存在
