获取征信报告-借么

说明

调用该接口可以根据爬虫任务ID获取获取用户的人行征信报告。

URL

http://<project>/h5/pbccrc/getReportJieMo

格式

JSON

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
renhangzhengxinTaskId false String 人行征信唯一标识

注意事项

例子

说明：调用本接口，获取用户的征信报告。

请求：https://uatapitxboss.txtechnologies.com:9018/h5/pbccrc/getReportJieMo

creditCardInfo->sixtydaysOverdraftMonth String 信用卡明细->准记卡账户有超过60天逾期记录月数
creditCardInfo->delqL5y90dayAmt String 信用卡明细->最近五年90天以上逾期次数
creditCardInfo->cancellDate String 信用卡明细->销户年月
creditLoanInfo->queryID String 贷款明细->编号

creditLoanInfo->financeCorporationBankName String 贷款明细->发放贷款机构名称，发放贷款银行的名称
creditLoanInfo->isDelq String 贷款明细->是否发生过逾期
creditLoanInfo->isCloseout String 贷款明细->贷款是否结清，购房贷款是否已结清
creditLoanInfo->transfer String 贷款明细->是否已转出
creditLoanInfo->badDebts String 贷款明细->是否已变成呆账
creditLoanInfo->releasedate String 贷款明细->发放日期
creditLoanInfo->deadline String 贷款明细->到期日期
creditLoanInfo->dueDate String 贷款明细->截至年月
creditLoanInfo->releaseCount String 贷款明细->贷款合同金额
creditLoanInfo->balance String 贷款明细->贷款余额
creditLoanInfo->overDueAmount String 贷款明细->逾期金额

creditLoanInfo->delqL5y90dayAmt String 贷款明细->最近五年90天以上逾期次数
creditLoanInfo->settleDate String 贷款明细->结清年月
