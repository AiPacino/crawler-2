前置规则字段说明

前置规则字段说明对照表：
字段 字段类型 字段说明
key String 业务唯一标识
taskId String 爬虫任务ID
name String 姓名
id_card String 身份证号码
in_time String 开户时长（秒）
call_time String 按照每个月为单位，统计每个月的主叫时长（秒）
interactMth String 月份
dialTime String 主叫时长（秒）
type String 业务类型（运营商）
