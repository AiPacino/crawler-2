获取征信报告

说明

调用该接口可以获取用户的征信报告。

URL

http://<project>/h5/v2/pbccrc/getCredit

格式

JSON

HTTP请求方式

POST

请求参数

# 必选 类型及范围 说明
username true String 用户名
password true String 登录密码
verifycode true String 登录页图片验证码
cookies true String 登陆页的cookie
tradecode true String 手机验证码（身份验证码）
key true String 业务唯一标识
owner true String 所属人

注意事项

    1.登录页的cookies以及图片验证码可以从获取登陆页接口获得。
    2.获取登录接口返回结果的cookies类型为Set<Cookie>而该接口cookies参数类型为String，故可利用JSON.stringify函数进行参数类型的转换。

例子

说明：非调试模式下调用本接口，获取人行征信登陆页的cookie，并进行人行征信系统登录页图片验证码的识别

请求：https://uatapitxboss.txtechnologies.com:9018/h5/v2/pbccrc/getCredit

返回：JSON
