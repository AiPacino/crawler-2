贷乎-前置规则

说明

该接口为推送前置规则

格式

JSON

HTTP请求方式

POST

注意事项

例子

说明：调用贷乎的接口，推送前置规则，返回推送结果状态

前置规则：JSON

JSON示例

{
	"taskID": "305af794-b1a3-45b3-8754-9e7d789e1b97",
	"name_on_card": "张三",
	"incomeAvg6": "null"
}

前置规则字段说明

前置规则字段说明对照表：
字段 字段类型 字段说明
taskID String 爬虫任务ID
name_on_card String 网银认证姓名
incomeAvg6 String 网银近6个月月均收入
