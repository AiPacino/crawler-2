学信网登录接口

说明

该接口通过输入的手机号、身份证、邮箱及密码，登录学信网，返回是否成功信息，数据格式为JSON

URL

http://<project>/xuexin/login

格式

JSON

HTTP请求方式

POST

请求参数

# 必选 类型及范围 说明
username true String 用户名
password true String 登录密码
taskid true String 任务id

例子

说明：账号登录学信网

请求：http://10.167.202.231:9150/xuexin/login

返回：JSON

返回字段

字段名 类型 注释
id String 唯一标识

cookies String cookies
phase String 当前步骤
phase_status String 步骤状态
description String 描述
finished Boolean 爬虫任务是否全部完成
error_code Integer 错误代码
error_message String 错误信息
etltime Date 统一数据时间
param String 爬取中需要的参数
param String 爬取中需要的参数
loginName String 登录名
crawlerHost String 爬虫节点的IP或hostname
crawlerPort String 爬虫节点的端口
webdriverHandle String
loginInfo String 登录信息
education_status Integer 学历信息状态
school_status Integer 学籍信息状态
