业务参数

说明

请求H5地址的业务参数

URL

http://<project>/h5/auth?owner=tianxi&key=testkey&redirectUrl=http%3a%2f%2ftest.harmonycredit.cn%2fhcweixin%2fcallback%2furlcallback.do%3fphoneNo%3dPHONENO%26item%3doperator

格式

无

HTTP请求方式

GET

请求参数

# 必选 类型及范围 说明
key true String 业务唯一标识
owner true String 所属者
redirectUrl false String 回调地址
username false String 姓名
mobilenum false String 手机号码
idnum false String 身份证号码

注意事项

  1. owner为固定值，比如：huicheng、xinhe、...
  2. redirectUrl为用户完成授权之后需要跳转页面的url地址

例子

说明：访问H5页面的业务参数

请求：https://uatapitxboss.txtechnologies.com:9018/h5/auth?key=test&owner=tianxi&redirectUrl=http%3a%2f%2ftest.harmonycredit.cn%2fhcweixin%2fcallback%2furlcallback.do%3fphoneNo%3dPHONENO%26item%3doperator
