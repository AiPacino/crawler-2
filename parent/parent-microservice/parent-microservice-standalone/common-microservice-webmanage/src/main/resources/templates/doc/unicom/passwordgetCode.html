修改密码验证获取接口

说明

该接口通过任务ID，获取修改用户密码的验证码，返回是否成功信息，数据格式为JSON

URL

http://<project>/passwordgetCode

格式

JSON

HTTP请求方式

POST

请求参数

# 必选 类型及范围 说明
usrnum true String 用户手机号
certNum true String 身份证号后六位
mobileMsg true Integer 手机验证码
newpassword true String 新密码
user_id true Integer 用户id
task_id true String 任务id

注意事项

  1. 手机号必须为11位手机号
