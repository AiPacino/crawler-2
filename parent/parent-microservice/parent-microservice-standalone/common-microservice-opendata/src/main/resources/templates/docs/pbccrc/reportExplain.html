人行征信数据报告

说明

调用该接口可以根据用户的taskid获取对应的人行征信报告。

URL

http://<project>/opendata/data/pbccrc/report/{taskid}

格式

JSON

HTTP请求方式

POST/GET

请求参数

# 必选 类型及范围 说明
taskid true String 任务号

注意事项

  1. taskid不能为空

例子

说明：利用taskid获取完成数据
